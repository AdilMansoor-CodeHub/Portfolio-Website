Description

OpenGL Physics Engine and Renderer in C++

Overview

This project involves developing a physics engine and renderer in C++ using modern OpenGL techniques. The engine implements essential game engine concepts like rigidbodies and gravity, supports the import of 3D models via the Assimp library, and features a user interface created with ImGui.

Core Features

  • Physics Engine:
    • Rigidbodies: Implemented rigidbody physics to simulate realistic object behavior, including collisions and interactions.
    • Gravity: Added gravity to affect objects, creating a realistic environment where objects fall, collide, and settle naturally.
  • OpenGL Renderer:
    • 3D Model Importing: Utilized the Assimp library to load and render various 3D model formats, enabling complex scenes with detailed objects.
    • Lighting and Shading: Integrated with the physics engine, the renderer uses advanced lighting techniques to enhance visual fidelity, including support for directional, point, and spotlight types.
  • UI with ImGui:
    • Interactive Interface: Implemented ImGui for a flexible and responsive user interface, allowing real-time adjustments to physics parameters, object properties, and rendering settings.
    • Debugging Tools: Includes tools for debugging, such as visualizing bounding boxes, collision shapes, and performance metrics.

Conclusion

This project represents a comprehensive effort to build a robust and flexible game engine foundation in C++. By integrating physics, rendering, and UI systems, the engine not only demonstrates key game development concepts but also offers a powerful platform for future projects and enhancements.
