Building a Modern 3D Renderer in C++ with OpenGL

Overview

This project showcases a 3D renderer I developed in C++ using modern OpenGL technologies. The renderer is designed to handle complex 3D models, advanced lighting techniques, and a variety of OpenGL effects, offering both visual fidelity and performance optimization.

Key Features

  • Modern OpenGL: Utilizes Vertex Array Objects (VAOs), Framebuffer Objects (FBOs), and Shader Storage Buffer Objects (SSBOs) to maximize performance and flexibility.
  • 3D Model Rendering: Supports loading and rendering of various 3D model formats with the Assimp library, breaking models into meshes, materials, and textures.
  • Advanced Lighting: Implements directional, point, and spotlight types using the Phong lighting model for realistic ambient, diffuse, and specular effects.
  • Post-Processing Effects: Includes bloom, depth of field, motion blur, and HDR with tone mapping for enhanced visual realism.
  • Optimizations: Features frustum culling, Level of Detail (LOD), and batch rendering to ensure high performance without compromising on quality.
