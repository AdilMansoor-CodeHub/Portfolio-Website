Amazon Home Page Frontend Clone: A Pixel-Perfect Recreation

Overview

Created a frontend clone of the Amazon home page. This project was all about learning fundamentals of web Development, aiming to replicate the look and feel of one of the most iconic e-commerce websites on the internet.

Key Features

  • Accurate replication of Amazon's home page layout and design.
  • Responsive design that adjusts seamlessly to different screen sizes.
  • Recreated key elements such as the navigation bar, product showcases, and footer.

Development Details

This clone was built using HTML, CSS, and JavaScript.
